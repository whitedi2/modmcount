Документ предназначен для ввода остатков по счетам на начало планового периода.

Вкладки Сырье, Кредиты и Покупатели доступны только для просмотра, т.к. редактирование данных остатков происходит из соответствующих дорументов.

Вкладка прочие доступна для редактирования. Причем для ускорения процесса можно воспользоваться кнопкой "Синхронизировать остатки". В этом случае будет опрошены и внесены в табличную часть документа фактические остатки по счетам.

Счета, остатки по которым можно синхронизировать автоматически:

68, 70, 75.1, 75.2, 75.3, 82, 83.1, 83.2, 97, 98.
