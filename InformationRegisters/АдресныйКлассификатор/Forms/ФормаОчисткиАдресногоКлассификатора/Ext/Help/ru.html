Удаление адресных сведений по выбранным адресным объектам из адресного классификатора.

Для очистки необходимо в списке адресных объектов отметить те, адресные сведения по которым планируется удалить.

Командная панель формы содержит следующие кнопки:

  • Очистить - удалить адресные сведения по выбранным объектам; в случае если регионы выбраны будет предложен диалог, в котором необходимо подтвердить удаление адресной информации
  • Отмена - отменить очистку адресного классификатора
  •  - эта справка

    • ВНИМАНИЕ: при очистке удаляются все записи по выбранным адресным объектам из информационной базы
